# MyWeddingDayPR

**MyWeddingDayPR** es una colección de invitaciones digitales para bodas, creada para parejas que desean compartir los detalles de su gran día de una manera moderna, elegante y sostenible.

Cada invitación ofrece una experiencia visual única, cuidadosamente diseñada para reflejar distintos estilos y personalidades. Desde lo clásico y romántico hasta lo tropical y minimalista, hay una opción para cada historia de amor.

## ¿Por qué una invitación digital?

- **Moderna y práctica** — se comparte con un solo enlace, desde cualquier dispositivo.
- **Sostenible** — sin papel, sin impresión, sin desperdicios.
- **Personalizable** — nombres, fecha, lugar, paleta de colores y más.
- **Cuenta regresiva en tiempo real** — los invitados saben exactamente cuánto falta.
- **Sin costo de envío** — llega al instante a todos tus invitados.
